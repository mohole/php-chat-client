# php-chat-client
JS Chat client related to PHP-02 lessons.

---

### Install
```bash
# step 1
git clone https://github.com/mohole/php-chat-client.git
# step 2
cd php-chat-client
# step 3
npm install
# step 4
npm start
```

Edit this line in the `js/utils.js` file to match your localhost settings
```javascript
const CHAT_API = 'http://localhost:3004';
```
